# 🧩 Systems Design & Architecture

A collection of structured system-analysis projects completed using industry frameworks such as UML, BPMN, CRUD matrices, SSDs, and structured requirements gathering.

These projects demonstrate:
- Modeling complex system logic
- Converting business needs into technical diagrams
- Producing development-ready specifications
- Understanding of system behavior, actors, and interactions

### 📂 Projects

### 🔹 RanHome — Full System Analysis & Design
**Tools:** UML, BPMN, Object-Oriented Analysis & Design  
**Description:** End-to-end system analysis and design for RanHome, covering the full modeling lifecycle from use case definition through to system sequence diagrams and CRUD mapping.  
**Artifacts included:**
Activity Diagram · Use Case Diagram · Domain Model Class Diagram · State Machine Diagram · Use Case Descriptions · Activity Diagram for Use Case · Systems Sequence Diagram · Use Case & CRUD Matrix
